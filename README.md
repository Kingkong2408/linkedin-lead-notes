# LinkedIn Lead Notes

LinkedIn Lead Notes is a browser extension that lets users save private notes on LinkedIn profiles and export saved leads to CSV.

## Features

- Save notes directly on LinkedIn profile pages
- Add a simple lead status
- Reopen a profile and see your saved note
- Export saved leads to CSV
- Store data locally in the browser
- No account required

## Data storage

Saved notes are stored locally in the user's browser using browser extension storage.

The extension does not send saved notes to an external server.
